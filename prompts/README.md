# 合同审阅提示词系统

## 概述

本目录包含合同审阅系统使用的提示词文件，用于指导AI模型进行专业的合同审查工作。

## 文件结构

```
prompts/
├── README.md                           # 本说明文件
└── contract_reviewer_prompt.txt        # 合同审查律师提示词
```

## 提示词文件说明

### contract_reviewer_prompt.txt

这是主要的合同审查律师提示词文件，包含：

- **角色定义**: 专业的合同审查律师
- **技能描述**: 合同审查、风险识别、条款分析等
- **行为准则**: 专业性、客观性、实用性等原则
- **工作流程**: 系统性的合同审查步骤
- **输出格式**: 标准化的修改点格式要求
- **示例说明**: 具体的修改点示例

## 使用方法

### 1. 直接使用

系统会自动从 `prompts/contract_reviewer_prompt.txt` 文件加载提示词：

```python
# 系统会自动尝试加载提示词文件
# 如果文件不存在，会使用默认的简化提示词
```

### 2. 自定义提示词

您可以修改 `contract_reviewer_prompt.txt` 文件来自定义提示词：

1. 编辑文件内容
2. 保持格式要求不变
3. 重新运行系统

运行测试脚本验证提示词文件是否正确：

## 输出格式要求

系统要求AI模型按照以下格式输出修改点：

```
【修改点X】- 修改点编号和标题

【原文】
原始条款内容...

【风险分析】
法律风险分析...

【修改后的内容】
建议的修改内容...
```

## 注意事项

1. **文件编码**: 确保提示词文件使用UTF-8编码
2. **格式保持**: 修改提示词时保持原有的格式结构
3. **示例更新**: 可以根据需要更新示例内容
4. **兼容性**: 系统会自动处理文件不存在的情况

## 故障排除

### 问题1: 提示词文件未找到
- 检查文件路径是否正确
- 确认文件名拼写正确
- 系统会自动使用默认提示词

### 问题2: 编码问题
- 确保文件使用UTF-8编码保存
- 检查文件是否包含特殊字符

### 问题3: 格式错误
- 检查提示词格式是否符合要求
- 确保包含必要的输出格式说明
